# LeGoodAnfitrion

Crea y organiza experiencias memorables para tus invitados.

LeGoodAnfitrion es una web/app en evolución para ayudar a anfitriones a planificar mejor sus eventos, gestionar invitaciones y personalizar cada experiencia según los gustos de sus invitados.

Web oficial:
[https://legoodanfitrion.com](https://legoodanfitrion.com)

## Qué propone LeGoodAnfitrion

- Planificación de eventos de forma simple
- Gestión de invitados e invitaciones RSVP en un solo lugar
- Perfiles enriquecidos para conocer mejor a cada invitado
- Sugerencias prácticas para ser un mejor anfitrión

## Tipos de experiencias

- Celebraciones
- Barbacoas
- Reencuentros
- Citas
- Clubes de lectura
- Quedadas informales con familia, amistades o compañeros

## Visión de producto

LeGoodAnfitrion nace de una idea clara:
ser buen anfitrión también se puede entrenar con ayuda de la tecnología.

Pilares del producto:

- Usabilidad para personas no técnicas
- Recomendaciones útiles y accionables
- Respeto por la privacidad y el consentimiento
- Experiencia multilingüe

## Estado

Proyecto en fase MVP y mejora continua.
Nuevas funcionalidades y mejoras de experiencia se publican de forma iterativa.
